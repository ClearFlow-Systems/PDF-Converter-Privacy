2 How the App Works

All processing happens entirely on your device:

  • You select PDF files from your computer using the system file picker.
  • The app renders each PDF page to an image using the bundled PDFium library.
  • Each image is passed to the bundled Tesseract OCR engine, which runs locally on your CPU.
  • Recognised text is written to an output file in a folder you choose.
  • No step in this process involves a network connection, a remote server, or any external service.

The app requests broadFileSystemAccess permission on Windows solely so you can open PDF files from anywhere on your system (network drives, external drives, etc.) and save output to any folder you choose. This permission is used only when you explicitly select files or folders via the system dialogs.

3 Network Access

PDF OCR Converter makes no network requests at runtime.

Tesseract OCR and all required libraries are bundled inside the application package. The app does not call home, check for updates automatically, or connect to any server while processing your documents.

The internetClient capability is not declared in the app manifest, which means Windows does not grant it outbound network access.

4 Third-Party SDKs and Analytics

This app contains no analytics SDKs, no advertising SDKs, and no tracking libraries of any kind.

The open-source components bundled with the app (Tesseract OCR, PDFium, pypdfium2, pytesseract, Pillow, python-docx, reportlab) operate entirely locally and do not transmit any data. Full license information for these components is available in the LICENSES.txt file included with the application.

5 Children's Privacy

PDF OCR Converter does not collect any data from anyone, including children under the age of 13. The app has no account registration, no online features, and no mechanism to collect personal information from any user of any age.
